### SpringMVC概述
![](../../.vuepress/public/img/59bbce50.png)
   
SpringMVC实现了视图代码和后台代码分离，同时提供了大量Web请求功能组件
1. 一个请求对应一个方法
2. 数据的自动绑定
3. 对JSON数据的支持
4. 对上传功能的支持
5. 对国际化的支持
6. 对Restful的支持
