### 微服务监控 Spring Boot Admin
Spring Boot Admin用于管理和监控一个或多个Spring Boot程序。Spring Boot Admin分为Server端和Client端，Client端可以通过Http向Server端注册，也可以结合Spring Cloud的服务注册组件Eureka进行注册。Spring Boot Admin提供了用AngularJs编写的UI界面，用于管理和监控。其中监控内容包括Spring Boot的监控组件Actuator的各个Http节点，也支持更高级的功能，如Turbine、Jmx、Loglevel等
