### Nginx常用命令
使用nginx操作命令前提条件：必须进入nginx的目录
```
/usr/local/nginx/sbin
```
    
查看nginx的版本号
```
./nginx -v
```
    
启动nginx
```
./nginx
```
    
关闭nginx
```
./nginx -s stop
```
    
重新加载nginx
```
./nginx -s reload
```
